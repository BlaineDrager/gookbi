# padding
* padding 속성은 ***`요소의 내부 여백을 지정하기`*** 위한 속성이다
* 값
    * 크기 값
* 형태
    * [값1] : `모든 방향에 값1` 을 적용한다.
    * [값1 값2] : `상하단에 값1 좌우에 값2` 를 적용한다.
    * [값1 값2 값3 값4] : `상, 우, 하, 좌 순서대로 값1, 값2, 값3, 값4`를 적용한다
* padding 속성을 적용한 내부 여백은 트리거가 발동해 다른 속성들이 요소에 적용된다
